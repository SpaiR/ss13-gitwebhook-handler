Labels:

Labels are used by auto-labeling feature. All fields are names of labels, which will be PR's marked.
Full description below:

  • Invalid Changelog - for PR with invalid 'in body' changelog.
  • Map Changes - handler automatically check PR diff and react on 'dmm' files changes with this label.
  • Icon Changes - same as for Map Changes, but reaction on 'dmi'.
  • Work In Progress - PR with [WIP] in title will be marked with this.
  • Do Not Merge - same as Work in Progress, but for [DNM].
    •
    About 'Class/Label' table. Label names to add are also taken from 'in body' changelog, by checking changes classes.
    This table gives ability to name any change class as you like. If class label name is empty, no label for this class will be added.
    Table is constructed from 'Available classes' list in 'CHANGELOG' tab.
